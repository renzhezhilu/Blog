##### 01.全局安装vue-cli
```javascript
npm install vue-cli -g
```
> 如报错无权限，加上sudo，输入密码继续
```javascript
sudo npm install vue-cli -g
```
> 其它一些命令
```javascript
vue -V //版本
vue -h //帮助
vue list //官方模板
```
